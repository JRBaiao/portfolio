Projects

AI-powered tools spanning agent development, adversarial security research, and red-team training simulations.

AI Agent — LLM Output Capture Tool

An AI-powered agent to capture, store, and organise outputs from LLM interactions, integrating prompt engineering and REST API integration.

AI Agents REST API Prompt Engineering

Built to systematically capture and structure outputs generated during LLM interactions, this tool integrates directly with REST APIs to persist responses for later analysis, auditing, or downstream automation — reducing the manual effort of tracking prompt/response pairs across sessions.

Prompt Injection Simulator

An interactive educational simulator demonstrating six AI security attack types against a Claude-backed LLM, with toggleable defenses and session logging.

Prompt Injection TF-IDF Logistic Regression

Covers Direct, RAG, and Agent attack categories against a Claude-backed LLM, with toggleable defence mechanisms and session logging. Implemented a detection layer using TF-IDF and Logistic Regression trained on the deepset/prompt-injections HuggingFace dataset, with a roadmap toward DistilBERT fine-tuning and SHAP explainability in subsequent phases.

Pretexting Script Generator & Red-Team Evaluator

A security training tool using a dual-LLM architecture: one model generates pretexting scripts, a second scores them as a red-team analyst.

Dual-LLM Red-Team Responsible AI

One model generates realistic pretexting scripts targeting corporate personas (IT helpdesk, HR, executives), while a second model acts as a red-team analyst, scoring each script on plausibility, psychological tactic, risk level, and detection likelihood. Outputs are scoped exclusively to security training scenarios through a responsible AI framework enforced at the system prompt level.
